1° Principio — Sistema Aperto — Regime Stazionario

Equazione Generale dell'Energia (Eulero)

Tra due sezioni di un sistema aperto in regime stazionario:

\[ q_{12} - l_{i,12} = (h_2 - h_1) + \frac{w_2^2 - w_1^2}{2} + g(z_2 - z_1) \]

q₁₂ = calore specifico scambiato [J/kg]  |  lᵢ = lavoro tecnico specifico [J/kg]  |  h = entalpia specifica = u + pv [J/kg]

w = velocità [m/s]  |  g·z = energia potenziale [J/kg]

Grandezza chiave

Entalpia Totale (o di Ristagno)

Si definisce l'entalpia totale come:

\[ h_{tot} = h + \frac{w^2}{2} + gz \]

In forma di potenza (per portata massica ṁ):

\[ \dot{Q} - \dot{P}_i = \dot{m} \left[ (h_2 - h_1) + \frac{w_2^2 - w_1^2}{2} + g(z_2 - z_1) \right] \]
Casi pratici

Casi Particolari

Ipotesi Equazione semplificata
Macchina adiabatica (q = 0), Δz ≈ 0 \( l_i = h_1 - h_2 \)
Scambiatore di calore (lᵢ = 0, Δw ≈ 0, Δz ≈ 0) \( q = h_2 - h_1 \)
Fluido incomprimibile (Bernoulli generalizzato) \( l_i = \frac{p_1 - p_2}{\rho} + \frac{w_1^2 - w_2^2}{2} + g(z_1 - z_2) \)
Ugello adiabatico (lᵢ = 0, Δz ≈ 0) \( h_1 + \frac{w_1^2}{2} = h_2 + \frac{w_2^2}{2} \)
